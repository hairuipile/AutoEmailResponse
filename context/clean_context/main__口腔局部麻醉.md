# 口腔局部麻醉  
|        | 利多卡因 | 普鲁卡因 | 布比卡因 | 丁卡因 | 碧蓝麻（阿替卡因） |
|--------|----------|----------|----------|--------|--------------------|
| 分类   | 酰胺类   | 脂类     | 酰胺类   | 脂类   | 酰胺类             |
| 注意点 |          | 药敏     |          | 药敏   | 慢性疾病、儿童慎用 |  
1. 下牙槽神经阻滞麻醉
- 进针点：颊脂垫尖，翼下颌皱褶中点外侧 0.3 - 0.4 cm，下颌牙
- 麻醉范围：下颌  
1. 舌神经阻滞麻醉
- 下牙槽神经阻滞麻醉注射针退出 1 cm，注射麻药
- 麻醉范围：同侧舌侧牙龈  
1. 颊神经
- 舌神经阻滞麻醉后退至肌层黏膜下
- 麻醉范围：下颌磨牙颊侧牙龈，颊部黏膜，肌肉
