# 阴道异常分泌物鉴别  
| \            | 特征                                 |
|--------------|--------------------------------------|
| 滴虫性阴道炎 | 稀薄脓性黄绿色泡沫分泌物、草莓样宫颈 |
| 霉菌性阴道炎 | 粘稠凝乳样、豆渣样分泌物             |
| 细菌性阴道炎 | 线索细胞 (+)、氨臭味分泌物           |
| 急性子宫颈炎 | 宫颈管棉拭子脓性分泌物、宫颈脆性增加 |
| 盆腔炎性疾病 | 宫颈举痛、附件区压痛                 |
