# 恐慌障碍 (Panic Disorder)

恐慌障碍是一种常见的焦虑障碍，特征是反复发作的、不可预测的惊恐发作。

## 什么是惊恐发作？

惊恐发作是突然出现的强烈恐惧或不适感，通常在几分钟内达到高峰，伴随多种身体症状。

## 症状

### 核心症状
- 突然感到强烈的恐惧或不适感
- 症状迅速达到高峰（通常在10分钟内）

### 身体症状
- 心悸、心跳加速或心率不齐
- 出汗
- 颤抖或发抖
- 气短或窒息感
- 胸痛或胸部不适
- 恶心或腹部不适
- 头晕、步态不稳、头重脚轻或昏厥感
- 发冷或发热感
- 感觉异常（麻木或刺痛感）
- 现实解体（感觉不真实）或人格解体（感觉与自己分离）
- 害怕失去控制或"发疯"
- 害怕 dying

## 诊断标准

根据DSM-5，诊断恐慌障碍需要满足以下条件：

1. 反复发作的不可预测的惊恐发作
2. 至少在发作后1个月内存在以下1项或多项：
   - 持续担心或焦虑再次发作
   - 对发作相关的症状或后果产生显著的行为改变
3. 这些症状不能归因于物质（如药物滥用、药物治疗）的生理效应或其他躯体疾病
4. 这些症状不能用其他精神障碍来更好地解释

## 病因

恐慌障碍的病因可能包括：

- **遗传因素**：有家族史的人更容易患病
- **神经生物学因素**：大脑中的杏仁核过度活跃，血清素等神经递质失衡
- **心理因素**：焦虑敏感性高、认知偏差
- **环境因素**：压力事件、创伤经历

## 治疗方法

### 心理治疗
- **认知行为疗法 (CBT)**：识别和改变负面思维模式，学习应对技巧
- **暴露疗法**：逐渐暴露于恐惧的情境或身体感觉
- **放松训练**：深呼吸、渐进性肌肉放松

### 药物治疗
- **抗抑郁药**：SSRIs（如帕罗西汀、舍曲林）是首选
- **苯二氮卓类药物**：在急性发作时可短期使用，但长期使用可能导致依赖
- **β受体阻滞剂**：可缓解身体症状

### 自助策略
- 学习识别惊恐发作的早期迹象
- 练习深呼吸和放松技巧
- 避免咖啡因、酒精和尼古丁
- 定期锻炼
- 保证充足的睡眠

## 就医指引

- 如果您频繁经历惊恐发作，影响了日常生活，请及时寻求专业帮助
- 精神科医生或心理治疗师可以提供准确的诊断和治疗方案
- 不要自行停药或更改治疗计划

## 预后

通过适当的治疗，大多数恐慌障碍患者可以显著改善症状，甚至完全康复。早期干预和持续治疗是关键。
