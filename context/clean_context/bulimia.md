# 神经性贪食（Bulimia Nervosa）

## 概述
以反复暴食发作与代偿行为（催吐、滥用泻药/利尿剂、过度运动等）为特征，患者对体重与体型过度关注，常伴随羞耻与罪责感。

## 诊断要点（DSM-5 摘要）
- 反复暴食（短时间内摄入大量食物，伴失控感）。
- 反复不适当的代偿行为以预防体重增加。
- 暴食与代偿行为平均每周至少一次，持续 3 个月。
- 自我评价过度受体型与体重影响。
- 不在神经性厌食的发作期间出现。

## 并发症与风险
- 电解质紊乱与牙齿损害（反复催吐）
- 胃肠道损伤与睡眠/情绪问题
- 共病：抑郁、焦虑、物质使用等

## 评估工具
- EDE-Q、SCOFF
- 医学评估：电解质、牙科检查等

## 治疗方法
### 心理治疗
- CBT-E：核心干预，针对暴食与代偿的维持因素
- 人际心理治疗（IPT）：处理关系冲突与情绪调节
### 医疗与营养
- 规范饮食与代偿行为管理
- 医疗监测与并发症处理
### 综合支持
- 家庭与同伴支持，减少羞耻与隐匿
- 正念与情绪调节训练

## 资源与热线
- 全国心理危机干预热线：`400-161-9995`
- 紧急帮助页：参见《紧急帮助》（`emergency.md`)

---

神经性贪食需要心理与医疗联合干预，关注共病与功能恢复。 
## 就医路径与患者自助清单
- 就医路径：医学评估并发症 → CBT-E/IPT → 规范饮食与代偿管理 → 并发症与共病随访
- 自助清单：暴食/代偿触发记录、规范餐单、情绪调节与正念练习、支持网络与复发预防
