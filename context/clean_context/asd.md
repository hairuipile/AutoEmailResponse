# 自闭症谱系障碍（Autism Spectrum Disorder, ASD）

## 概述
ASD 主要表现为社会交往与沟通困难，以及受限、重复的行为与兴趣。谱系范围广，个体差异显著，常伴随感觉处理差异、学习困难或特殊才能。

## 诊断标准（DSM-5 摘要）
- 多场景的社会沟通与互动缺陷（非语言沟通、建立与维持关系等）。 
- 受限与重复的行为、兴趣或活动（刻板动作、强烈执着、感官反应异常等）。
- 症状在早期发育阶段出现并导致功能损害。
- 结合智力与语言水平分级。

## 评估工具
- ADOS-2、ADI-R（专业结构化评估）
- CARS、SRS-2（量化评估）
- 共病筛查：焦虑、抑郁、ADHD 等

## 支持与干预
### 教育与康复
- 结构化教学（TEACCH）
- 应用行为分析（ABA）
- 语言与社交技能训练
### 医疗与综合支持
- 睡眠、消化、感觉处理等伴随问题管理
- 家庭支持与照顾者培训
- 学校与社区融合支持

## 特殊人群与伦理
- 尊重神经多样性，避免污名化，强调优势与需求并存。
- 个体化支持目标，以功能提升与生活质量为核心。
- 与家属、教育与医疗团队共同制定计划。

- 就医路径：专业评估（ADOS/ADI-R）→ 个体化教育与康复计划 → 医疗随访与家庭支持 → 学校/社区融合
