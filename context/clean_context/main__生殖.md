# 生殖
## 男性生殖功能与调节
0. 性器官
- 主性器官: 睾丸 (生精小管、间质细胞)
- 附属性器官
0. 功能
- 生精作用
- 内分泌作用
+ 间质细胞: 雄激素 (胆固醇->孕烯醇酮->雄激素)
+ 支持细胞: 雄激素结合蛋白 (ABP)、抑制素
0. 睾酮的生理作用
- 影响胚胎分化
- 维持生精作用
- 刺激附性器官的生长和维持性欲
- 促进蛋白质合成、骨骼生长、RBC的生成等
0. 下丘脑-垂体-睾丸轴
- 下丘脑->垂体->FSH (卵泡刺激素) ->生精小管
- 下丘脑->垂体->LH (黄体生成素) ->间质细胞->睾酮->雌二醇 (负反馈)
