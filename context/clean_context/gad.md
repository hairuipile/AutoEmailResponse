# 广泛性焦虑障碍 (Generalized Anxiety Disorder, GAD)

广泛性焦虑障碍是一种常见的慢性焦虑障碍，特征是过度、难以控制的担忧，持续至少6个月。

## 什么是广泛性焦虑障碍？

广泛性焦虑障碍患者会对多种生活领域（如工作、健康、家庭、财务）产生过度的担忧，这种担忧往往是不合理的，但患者无法控制。

## 症状

### 核心症状
- 过度的焦虑和担忧，持续至少6个月
- 难以控制这种担忧

### 身体症状
- 坐立不安或感觉紧张
- 容易疲劳
- 注意力难以集中或头脑一片空白
- 易怒
- 肌肉紧张
- 睡眠障碍（难以入睡、易醒或睡眠质量差）

### 心理症状
- 对日常事务过度担忧
- 担心失去控制
- 担心发生不好的事情
- 灾难化思维

## 诊断标准

根据DSM-5，诊断广泛性焦虑障碍需要满足以下条件：

1. 在至少6个月的多数日子里，对多种事件或活动（如工作、学校表现）过度焦虑和担忧
2. 难以控制这种担忧
3. 焦虑和担忧与下列6种症状中的至少3种（儿童只需1种）相关，且在至少6个月的多数日子里存在：
   - 坐立不安或感觉紧张
   - 容易疲劳
   - 注意力难以集中或头脑一片空白
   - 易怒
   - 肌肉紧张
   - 睡眠障碍
4. 焦虑、担忧或身体症状导致临床显著的痛苦或社交、职业或其他重要功能的损害
5. 这些症状不能归因于物质（如药物滥用、药物治疗）的生理效应或其他躯体疾病
6. 这些症状不能用其他精神障碍来更好地解释

## 病因

广泛性焦虑障碍的病因可能包括：

- **遗传因素**：有家族史的人更容易患病
- **神经生物学因素**：大脑中的杏仁核和前额叶皮层功能异常，血清素、去甲肾上腺素等神经递质失衡
- **心理因素**：完美主义、低自尊、灾难化思维
- **环境因素**：压力生活事件、童年创伤

## 治疗方法

### 心理治疗
- **认知行为疗法 (CBT)**：识别和改变负面思维模式，学习应对技巧
- **正念疗法**：学习活在当下，减少担忧
- **接受与承诺疗法 (ACT)**：接受焦虑，专注于有价值的行动

### 药物治疗
- **抗抑郁药**：SSRIs（如舍曲林、艾司西酞普兰）和 SNRIs（如文拉法辛）是首选
- **苯二氮卓类药物**：短期使用可缓解症状，但长期使用可能导致依赖
- **丁螺环酮**：非苯二氮卓类抗焦虑药，无成瘾性

### 自助策略
- 练习深呼吸和放松技巧
- 保持规律的运动
- 限制咖啡因和酒精摄入
- 保证充足的睡眠
- 学习时间管理和问题解决技巧
- 培养兴趣爱好，转移注意力

## 就医指引

- 如果您的焦虑和担忧影响了日常生活，请及时寻求专业帮助
- 精神科医生或心理治疗师可以提供准确的诊断和治疗方案
- 治疗可能需要数周或数月才能看到明显效果，请保持耐心

## 预后

通过适当的治疗，大多数广泛性焦虑障碍患者可以显著改善症状，提高生活质量。持续治疗和自我管理策略对于预防复发非常重要。
