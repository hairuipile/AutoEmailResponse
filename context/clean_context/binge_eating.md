# 暴食障碍（Binge-Eating Disorder）

## 概述
以反复暴食发作为特征，但不伴随常规的代偿行为（不同于神经性贪食）。暴食常伴随情绪困扰、羞耻与体重变化，并影响身心健康。

## 诊断要点（DSM-5 摘要）
- 反复暴食（出现失控感，进食速度快、食量大、独自进食、餐后内疚等）。
- 暴食平均每周至少一次，持续 3 个月。
- 明显痛苦与功能损害。
- 无规律性代偿行为。

## 评估与风险
- 体重、代谢与心血管风险评估
- 共病：抑郁、焦虑、睡眠障碍等
- 饮食与生活方式评估

## 治疗方法
### 心理治疗
- CBT-E：识别触发因素与建立规范饮食
- 人际心理治疗（IPT）：改善人际压力与情绪调节
- 正念与情绪调节训练
### 医疗与生活方式
- 营养指导与体重管理
- 睡眠与压力管理
- 必要时转诊内分泌评估

## 资源与热线
- 全国心理危机干预热线：`400-161-9995`
- 紧急帮助页：参见《紧急帮助》（`emergency.md`)

---

暴食障碍的核心在于情绪调节与规范饮食，联动心理与医疗支持可改善预后。 
## 就医路径与患者自助清单
- 就医路径：医学评估 → CBT-E/IPT → 营养与体重管理 → 共病与睡眠随访
- 自助清单：暴食触发记录、规范餐单与餐时、情绪调节技能、支持网络与复发预防
